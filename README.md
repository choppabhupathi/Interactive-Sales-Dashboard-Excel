# Interactive Sales Performance Dashboard (Excel) 📊✨

## 🚀 Project Overview
This project transforms raw electronics store sales data into an interactive visual dashboard. The goal is to analyze regional performance and sales representative efficiency using advanced Excel features.

## 🛠 Features & Tools Used
- **Data Cleaning:** Structured messy data using "Text to Columns" and formatting.
- **Pivot Tables:** Summarized large datasets to track revenue by Region and Category.
- **Pivot Charts:** Visualized trends for quick decision-making.
- **Interactive Slicers:** Added dynamic filters for 'Sales Person' and 'Category' to allow user-driven exploration.
- **Analytical Insights:** Identified top-performing zones and stock requirements.

## 📈 Key Insights
- The **South Region** showed the highest sales growth in the first week of January.
- **Laptops** continue to be the primary revenue driver across all regions.
- Real-time filtering helps managers monitor individual performance instantly.
